label
------
During pod creation
Syn - kubectl <label-name> <pod_name> --image=url 
Ex - kubectl run mypod --image=quay.io/gauravkumar9130/mywebapp

After POD creation
Syn - kubectl label pod pod_name key=value (pair of key value )
Ex - kubectl label pod mypod app=web
here app is the attribute/ key and web is the value


Selector
-------------------
Selector  - is used for selecting pods based on the labels.
ex - 
kubectl get pods --selector=app=web --show-labels
here app=web is the key value pair for labels.


Replica Set

To deploy a container with replicaset
use a yaml file for configuration - replica_set.yml
kubectl create -f replica_set.yml
--> this will create no pods for the container.

to stop all the pods which are replicated from relica_set.yml use the below command
 SYN - kubectl scale replicaset <replicaset-name> --replicas=0
 EX -  kubectl scale rs rs-web --replicas=0
 Note you can use replicaset or rs in the above command.
